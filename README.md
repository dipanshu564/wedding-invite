# Himanshu & Akanksha Wedding Invitation

A cinematic, luxury-level royal Indian wedding invitation website for **Himanshu & Akanksha**. This project features a rich royal navy and gold theme with advanced 3D animations, interactive elements, and a romantic aesthetic.

## ✨ Features

- **Royal Theme:** Deep navy blue and metallic gold palette with Mughal-inspired patterns.
- **Cinematic Hero:** Full-screen hero section with parallax effects and glowing typography.
- **3D Interactions:** 3D tilt cards, flip gallery, and a 3D animated venue pin.
- **Interactive Invitation:** A cinematic 3D unfolding animation for the invitation opening.
- **Romantic Accents:** Floating red love emojis, heart rain effects, and red sparkle particles.
- **Celebratory Effects:** Custom gold and crimson red confetti bursts.
- **Music:** Integrated background music with an animated equalizer control.
- **Mobile Friendly:** Responsive design with a sliding side navigation menu.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion (motion/react)
- **Icons:** Lucide React
- **Effects:** Canvas Confetti

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <project-directory>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

To start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create a production build:
```bash
npm run build
```
The optimized files will be generated in the `dist/` directory.

### Linting

To check for TypeScript errors:
```bash
npm run lint
```
